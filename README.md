# Java Banking System

A console-based banking application developed to practice core Object-Oriented Programming (OOP) and procedural logic in Java. This project provides a robust foundation for managing multiple user accounts with secure access.

## Key Features
- **Account Creation:** Dynamically create and store up to 20 client accounts with unique IDs.
- **Secure Authentication:** PIN-based login system for users with a 3-try limit for security.
- **Financial Operations:** 
  - Deposit and Withdrawal management.
  - Fund transfers between accounts with ID validation.
- **Admin Controls:** Administrator access to view all registered accounts.

## Tech Stack
- **Language:** Java
- **Environment:** Console-based interface
